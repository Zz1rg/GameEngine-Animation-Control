# Simple Animation Control

**Description:** This is a simple animation control code using OpenGL, we can control the character by
  - Hold **E** to **Walk**
  - Hold **R** to **Run**
  - Press **Space Bar** to Roll
  - If nothing is pressed then the **Idle** animation will be played
